# **App Name**: StudyBuddy AI

## Core Features:

- Content Ingestion: Allow users to upload study material such as notes, articles, and PDFs. Also supports importing of information via URLs.
- Intelligent Question Generation: Use generative AI to formulate relevant questions based on the uploaded content to quiz the user.
- Personalized Quiz Creation: Create quizzes with different formats such as multiple choice, true/false, and short answer based on user preference.
- Performance Tracking: Track the user's performance on quizzes and provide insights on areas of improvement. Display detailed analytics of the user's study habits and quiz scores.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire confidence and intellect.
- Background color: Light blue-gray (#E8EAF6), desaturated to 20% to ensure a calm reading experience.
- Accent color: Analogous purple (#7E57C2), brighter and more saturated, for highlights and interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a balance of modernity and warmth, suitable for both headings and body text.
- Code font: 'Source Code Pro' for displaying any code snippets that may appear in study materials.
- Use simple, outline-style icons for easy recognition.
- Use subtle transitions and animations for feedback.
